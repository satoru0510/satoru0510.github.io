Cubic（カスタムOSイメージ作成ソフト）

Ubuntuでの実行を前提とする。 Ubuntuへのインストール （参考） 
sudo apt-add-repository universe
sudo apt-add-repository ppa:cubic-wizard/release
sudo apt update
sudo apt install --no-install-recommends cubic

パスワードなしでsudoできるようにする（Ubuntu）

当然、セキュリティリスクが激増するため安易に行ってはいけない。
以下のコマンドで、sudo権限を編集する。
sudo visudo
ファイルに以下を追記する。
＜ユーザ名＞ ALL=NOPASSWD: ALL

NASをマウントする（参考）

マウントポイントを作成する
mkdir sambashare
cifs-utilsパッケージインストール
sudo apt install cifs-utils -y
マウント
sudo mount -t cifs -o noperm,username=＜NASのユーザー名＞,password=＜NASのパスワード＞ //＜NASのIPアドレス＞/＜共有フォルダ＞ sambashare
